# Todo App

A learning-focused todo list app.

**Stack:**
- [Convex](https://convex.dev) — backend + reactive database
- [Vite](https://vite.dev) + [React](https://react.dev) + TypeScript — frontend
- [Tailwind CSS](https://tailwindcss.com) — styling
- [shadcn/ui](https://ui.shadcn.com) — components (Radix UI under the hood)

## Getting started

```bash
npm install      # install dependencies
npm run dev      # start the Vite dev server (http://localhost:5173)
```

## Scripts

| Command | What it does |
|---------|--------------|
| `npm run dev` | Start the dev server with hot-reload |
| `npm run build` | Type-check and bundle for production |
| `npm run preview` | Preview the production build locally |
| `npm run lint` | Run ESLint |
